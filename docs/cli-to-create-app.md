## CLI

??? tip "Remember to install FletEasy first."
     ```bash
     pip install flet-easy[all]
     ```

!!! note
    If you use the [`fs`](/flet-easy/0.2.0/cli-to-create-app/) cli, it is important to have [`git`](https://git-scm.com/downloads) installed.

```bash
fs init
```
